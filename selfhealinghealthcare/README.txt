PROJECT NAME: AI-DRIVEN SELF-HEALING CLOUD FOR HEALTHCARE

DESCRIPTION:
This project monitors a system process and automatically restarts it if it stops. It demonstrates a self-healing system used in healthcare cloud environments.

TECHNOLOGIES USED:

* Python
* OS Library
* Task Scheduler

HOW TO RUN:

1. Open the project in VS Code
2. Run the file monitor.py
3. The script will monitor Notepad
4. If Notepad is closed, it will restart automatically

FILES:

* monitor.py → main script
* logs.txt → stores logs
* README.txt → project details
